flask_chatroom
==============
   重回十年前
===

+ 这是一个迷你聊天室，可以多人同时在线聊天,可以在多个房间切换。
+ 使用WebSocket协议，实现双向通信，具体采用[flask_socketio模块](http://flask-socketio.readthedocs.io/en/latest/)

---
基本功能：
+ 1 登陆
+ 2 多人即时在线聊天
+ 3 聊天室切换
